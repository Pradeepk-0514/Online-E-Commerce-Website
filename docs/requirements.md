# E-Commerce Platform 

## Project Name

Project Name: Angadiq

---

## Project Description

Build a multi-vendor e-commerce platform where customers can purchase products from multiple sellers and administrators can manage the entire platform.

---

## Technology Stack

### Frontend

* HTML
* CSS
* JavaScript
* TypeScript
* React

### Backend

* Java
* Spring Boot

### Database

* PostgreSQL

---

# User Roles

There are three roles in the system.

1. User (Customer)
2. Seller (Vendor)
3. Admin

---
